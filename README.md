# Naomi

Локальный чат-компаньон — живая тёплая собеседница. Просто болтает. Работает на **твоей подписке ChatGPT**, без платного API.

## Запуск

```sh
./run.sh        # → http://127.0.0.1:8765
```

Первый раз: ⚙️ → **Аккаунт** → «Войти через ChatGPT».

## Стек

- **`backend/`** — Python (FastAPI): запрос напрямую в подписку ChatGPT, личность Наоми в системном промпте.
- **`frontend/`** — React без сборщика (esbuild).
- **`naomi.md` / `pravila.md`** — личность и правила, правятся на лету.

Модель и уровень размышления — ⚙️ → **Агент** (по умолчанию `gpt-5.5` / `low`).

Подробности — ⚙️ → **Документация** (или [DOCUMENTATION.md](DOCUMENTATION.md)).
